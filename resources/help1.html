MapEdit - editor for PCRaster and GTiff raster maps

Version: 3.0
Date: 10 jan 2022

Loading maps

Mapedit works with two map layers: the base map is a reference map which is not changed, and the top map can be edited. When you load two maps, the first one you choose is the base layer/reference map, the second one is the edit layer. If the edit layer is a GeoTiff map, it will be saved as a tif file.

If you load only one map, this is assumed to be the base layer. An empty map is created, filled with the value '0' (zero) that can be edited and saved using you own name. If the base map is a tif file, the empty edit map will be a tif file with the ESPG information of the base layer, else it will be a PCRaster map file.

The legend of the base map is shown to the left of the map, the legend of the edit layer to the right.

Editing options

Edit shapes are selected with the tool buttons. After selecitng a shape, areas are digitized with the left mouse button. Single cells and rectangles have only one point (cell) or two points in opposiite corners. Lines and polygones needs to be stopped if you want to digitize a second line or polygon. You stop editing a shape by clicking with the right mouse button. If you then click with the left mouse button, a new shape is started.

Once you have digitized one or more cells or shapes, you can give the new edit value by clicking on the "v" next to the edit field. Changes are made and the digitizing is reset.

If you want to restire the original values, digitize one or more areas that you want to restore, close it, and click on the restire button (with tthe two round arrows).

Display options

The edit layer is semi-transparent on top of the base layer. The transparancy can be chgosen with the slider.

Both the base and edit layer can have different colors, click on the palette tool buttons to change the color setting of the maps and legends.

The slider influence the min and max display values, so you can show particular areas well.

You can zoom in and out with the mouse wheel. The reset size icon resets the zoom level to the full map.

You can drag the map by keeping the right mouse button pressed and dragging the map.

good luck!
