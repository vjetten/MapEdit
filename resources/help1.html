MapEdit - editor for PCRaster and GTiff raster maps

Version: 3.2.1
Date: 20 mar 2022

Loading maps

Mapedit works with two map layers: the base map is a reference map which is not changed, and the top map can be edited.

The transparanecy of the toplayer can be adjusted, and of the Base and Edit maps the min and max displayed and the palette can be adjusted.

There are 3 ways to use mapedit:

  • If you load two maps, the first one you choose is the base layer/reference map, the second one is the edit layer. If the edit layer is a GeoTiff map, it will be saved as a tif file.
    Both maps need to have the same nr. of rows, columns and cellsize
  • If you load only one map, this map is loaded TWICE: the base layer is seen for reference and it can also be edited. You are editintg in fact the base layer.
  • If you load maps and check the option "Edit an empty layer", the edit map is replaced with an empty map is created filled with the value '0' (zero) that can be edited and saved using you own name. Any edits you made will be lost!

If the base map is a GeoTiff file, the edit layer will be saved as GeoTiff file with the ESPG information of the base layer, else it will be a PCRaster map file (default).

The legend of the base map is shown to the left of the map, the legend of the edit layer to the right.

Editing options

Edit shapes are selected with the tool buttons. After selecitng a shape, areas are digitized with the left mouse button. Single cells and rectangles have only one point (cell) or two points in opposiite corners. Lines and polygones needs to be stopped if you want to digitize a second line or polygon. You stop editing a shape by clicking with the right mouse button. If you then click with the left mouse button, a new shape is started.

Once you have digitized one or more cells or shapes, you can give the new edit value by clicking on the "v" next to the edit field. Changes are made and the digitizing is reset.

If you want to restire the original values, digitize one or more areas that you want to restore, close it, and click on the restire button (with tthe two round arrows).

Display options

The edit layer is semi-transparent on top of the base layer. The transparancy can be chgosen with the slider.

Both the base and edit layer can have different legend colors, click on the palette tool buttons to change the color setting of the maps and legends (def. base layer in rainbow legend and edit layer in grey scale).

The sliders influence the min and max display values, to better show certain areas well.

Mouse wheel: zoom in and out. The reset size icon resets the zoom level to the full map.

Right Mouse button: drag the map by keeping the right mouse button pressed and dragging the map.

good luck!
